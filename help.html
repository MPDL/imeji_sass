Create a Collection

Note: Users require the appropriate authorization to create new collections. Please contact the support team with any question regarding your account.

  • 1. Go to the collections page and click on New collection
  • 2. Enter the required information (collection name, author name and organization). Information on further authors and a description of the collections content can be added optionally.
  • 3. Via the plus (and minus) buttons, further fields for authors and organizations can be added (or removed).
  • 4. Save.

Edit the Collection information

  • 1. Go to the collection you want to edit
  • 2. In the menu Collection, click on Edit information
  • 3. Modify the values you wish to update and save

Share a Collection

  • 1. Go to the collection you want to share
  • 2. In the menu Collection, click on Share
  • 3. Click on
    • a. Share with a user: Enter the email(s) (separated by a semicolon) of the user you want to share with
    • b. [If available] Share with a user group: Select one user group from the list
  • 4. Select the permission of the user(s) / user group:
    • a. Read: Can view and download the files, the metadata and the collection informations
    • b. Edit In addition to the Read permission, can upload and delete a file, edit metadata and the licenses of the item, edit the collection informations
    • c. Admin In addition to the Edit permission, can share, release and delete the collection
  • 5. Share

Release a Collection

Note: You can only release private collections. A release makes the collection and all included items visible to the public. Afterwards neither the collection nor the items can be deleted any longer, but they need to be discarded instead. The release does not freeze the collection completely, in particular new items can still be added.

  • 1. Go to the collection you want to release
  • 2. In the menu Collection, click on Release

Delete /Discard a Collection

  • 1. Go to the collection you want to delete / discard
  • 2. In the menu Collection, click on
    • [IF the collection is private] Delete: The collection, the items and the files are all deleted
    • [IF the collection is released] Discard: The files are deleted, but the collection and items remains, with the discard comment

Assign a DOI to a Collection

Note: You can only assign a DOI to a released collection and you can only assign one DOI per collection.

  • 1. Go to the collection you want to assign a DOI to
  • 2. In the menu Collection, click on Add DOI to collection
  • 3. Accept the Terms of Use
  • 4. You can now cite your collection with the assigned DOI

Cite a collection

  • 1. Go to the collection you want to cite
  • 2. Click on more, right under the collection title
  • 3. Copy the text Cite as.

Data quota exceeded

Please contact:
  • The support team if you are the creator of this collection
  • The owner of the collection, if the collection has been shared. You can find the owner of the collection on the share icon

For security reasons, imeji restrict the list of authorized file formats. To get more information, contact the support team

Your account is not authorized to create a new collection, and you don't have permission to upload files in at least one collection. Please aks the support team for the authorization to create a new collection

Describe item(s) with metadata

imeji offers you 3 different ways to describe your items with metadata:
  • a. Describe one item
  • b. Describe many items of a collection
  • c. Describe all items of a collection

Describe one item

  • 1. Go the item you want to describe with metadata
  • 2. Click on Edit
  • 3. Fill the form
  • 4. [If you want to add a new entry] Click on Add metadata:
    • a. Select a metadata from the list
    • b. [If you want to add a new metadata] Click on Add metadata at the bottom of the list and create a new metadata
  • 5. Save

Describe many items of a collection

  • 1. Go the collection containing the items you want to edit
  • 2. Select the items you want to edit
  • 3. In the menu X Selected Items, click on Edit
  • 4. Add metadata:
    • a. Edit a single cell, by clicking on to add a new value, or to remove a value
    • b. Add a value to a whole column, by clicking on
    • c. Remove all values of a column, by clicking on
    • d. Remove a column, by clicking on
    • e. Change the name of column by clicking on
    • d. Add a new column by clicking on Add column
  • 5. Save

Describe all items of a collection

  • 1. Go the collection containing the items you want to edit
  • 2. [If some items are selected], click on Select none
  • 3. In the menu X Items, click on Edit all
  • 4. Click on Select a metadata
    • a. Select a metadata from the list
    • b. [If you want to add a new metadata] Click on Add metadata at the bottom of the list and create a new metadata
  • 5. Enter a value
  • 6. Choose one method to add the metadata to the collection items:
    • a. Add a new value to all: The value will be append to all items of the collection, no matter whether the items have already a value for with this metadata
    • b. Add only if value is empty: The value will only be append to the items of the collection which don't have any value for this metadata
    • c. Overwrite all values: The value will be append to all items of the collection. Previous values for this metadata will be removed

Add license to item(s)

  • 1. Go the collection containing the items you want add the metadata
  • 2. Choose one the following option:
    • a. Go to the page of an item and click on Edit
    • b. Select some items and click on Edit all licenses in menu X Selected Items
    • c. Click on Edit all licenses in menu X Items (Ensure that no item is selected)
  • 3. Enter a a license:
    • a. Select one predefined license from the list
    • b. OR, click on
    • Enter your own license
      • and enter you license informations
  • 4. Save
    • a. [If the collection is private] Choose if you want to replace license of the items having already a license
    • b. [If the collection is released] the new license will be append to the item(s). To the previous license will be added a validity period.

Delete / Discard item(s)

  • 1. Go the collection containing the items you want to delete / discard
    • a. [If the collection is private] Choose one the following option:
      • i. Go to the page of an item and click on Delete
      • ii. Select some items and click on Delete in menu X Selected Items
      • iii. Click on Delete all in menu X Items (Ensure that no item is selected)
    • b. [If the collection is public] Choose one the following option:
      • i. Go to the page of an item and click on Discard
      • ii. Select some items and click on Discard items in menu X Selected Items

Move item(s) to another collection

Note: You can only move item(s) of a private collection.

  • 1. Go to the collection containing the items you want to move
    • a. Go to the page of an item and click on Move
    • b. Select some items and click on Move in menu X Selected Items
  • 2. Select the collection you want to move the item(s) to
  • 3. [If the target collection is released] Select a license to be added to the moved items

Simple Search

  • The simple search allows the user the search for all available metadata (item metadata, filename, collection information) as well as for the full text (in case of text document, like pdf or doc).
  • The simple search allows to search either for:
    • a. All items of all collections
    • b. All items of one collection (available only from collection page)
    • c. All collections.
  • The simple search is compatible with the imeji search syntax

Advanced Search

Note: The advanced search is restricted to items.

  • 1. Go to the simple search page
  • 2. Search for
    • a. All fields (item metadata, filename and optionally the fulltext)
    • b. Specific metadata
    • c. File types
    • d. Licenses
    • e. Technical metadata of files
Note: The advanced search produced query according to the imeji search syntax.
